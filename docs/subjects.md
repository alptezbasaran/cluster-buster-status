# Subject overlays

Every **clustered** set, browsable here — the headline **clusters-on-tracings**
overlay for each galaxy, the same view Zooniverse volunteers judge for good
spiral arms. Pick a set below; the status dot marks which are
<span class="cb-badge live">live</span> on Zooniverse versus
<span class="cb-badge clustered">clustered</span> and queued.

The anchor set `cluster-buster-001` is the known-good control: 500 galaxies at
`agreement 65 · correction 55 · ratio 1.5`. Filter by arm count, search a gid,
and click any overlay to enlarge. Cross-reference a galaxy via its SDSS, Legacy
Survey, or LEDA link.
